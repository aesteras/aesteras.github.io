This Website

Right now, this website is a custom implementation of a framework-less Single Page Application.

It uses the JavaScript Fetch API to get the content and the pushState() method of the History API for navigation.

It also has a responsive design, along with some JavaScript tricks and CSS styles which make possible things like the drop-down menu or the automatic repositioning of images.

If you wish, you are free to check the implementation in the source code. The files have not been minified or uglified, but be warned: HERE BE DRAGONS.
